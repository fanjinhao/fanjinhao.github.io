绝对值最小

题目详情:


给你一个数组A[n],请你计算出ans=min(|A[i]+A[j]|)(0<=i,j<n).                                                                        



例如：A={1， 4， -3}，

则：

|A[0] + A[0]| = |1 + 1| = 2.

|A[0] + A[1]| = |1 + 4| = 5.

|A[0] + A[2]| = |1 + (-3)| = 2.

|A[1] + A[1]| = |4 + 4| = 8.

|A[1] + A[2]| = |4 + (-3)| = 1.

|A[2] + A[2]| = |(-3) + (-3)| = 6.

所以ans=1.

输入描述：

有多组测数数据，每组数据有两行，第一行包含一个正整数n(0<n<=100000),第二行包含n个整数，分别表示A[0],A[1]，A[2],….,A[n-1]，（|A[i]|<2^30）。

输入以文件结束。

输出描述：

对于每组数据，输出相应的答案。

答题说明:

输入样例：

3

1 4 -3

1

2

3

-1 -2 -5

3

1 2 3

2

0 5
